# 北华大学猫猫网站（校园猫档案与社区）项目需求文档

## 1. 项目概述

## 1.1 项目背景

针对北华大学校内流浪猫/校园猫数量多、分布广的现状，建设一个集档案管理、信息共享、社区互动、AI娱乐于一体的数字化平台 。

## 1.2 核心目标

- ​**档案化管理**：为校园猫建立终身电子档案 。
- **社区化互动**：提供图片墙、评论、点赞等社交功能 。

## 3. 功能需求详述

## 3.1 猫猫档案管理模块 (Cat Resource)

该模块是系统的核心，需实现对“猫猫”这一资源的完整生命周期管理 。

- ​**档案展示**：支持按校区、品种、关键词检索，支持按热度或最新发布排序 。
- **详细信息**：展示包括性别（公/母/未知）、是否绝育、常驻地点、性格标签、浏览量及点赞数在内的深度数据 。
- ​**状态控制**：管理员可一键切换猫猫的显示/隐藏状态 。

## 3.2 首页与运营模块 (Display & Widgets)

- ​**数据汇总**：实时展示校猫总数、今日活跃猫数 。
- ​**今日最佳**：每日推荐一只特定的猫，展示上榜理由与推荐分值 。
- ​**轮播系统**：通过权重（Sort）控制首页 Banner 的展示顺序 。

## 3.3 社区互动模块 (Community)

- ​**内容发布**：用户可发布带有图片、关联特定猫猫 ID 的帖子或评论 。
- ​**互动反馈**：支持对猫猫、帖子、评论三个维度的点赞或取消点赞操作 。
- ​**层级评论**​：支持回复功能，通过 `parentId` 实现评论嵌套 。
